# QA Tester Skills

## Descripción
Habilidades de Quality Assurance y Testing para garantizar la calidad de software.

## Paths de Aprendizaje

### SQL para Testing
- SELECT y filtros básicos
- JOINs para validar relaciones  
- Subqueries para verificaciones complejas
- Funciones de agregación para reportes

### Python para Automatización
- Sintaxis básica
- Estructuras de control
- Manejo de archivos CSV/JSON
- Requests para API testing

### Playwright E2E
- Instalación y configuración
- Selectores y localización
- Acciones (click, type, scroll)
- Assertions
- Fixtures y hooks

### Testing Manual
- Tipos de testing (funcional, regresión, smoke)
- Creación de casos de prueba
- Reporte de bugs
- Pruebas exploratorias

## Herramientas
- Postman / Insomnia
- JIRA / Linear
- TestRail / Zephyr
- Browser DevTools

## Certificaciones
- ISTQB Foundation
